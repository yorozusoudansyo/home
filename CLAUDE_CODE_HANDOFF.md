## 3. LPの位置づけ・目的

このLPは以下のビジネスファネルの入口となる重要なページです。

```
Note・SNS発信（認知）
　　↓
LP（よろず相談所の説明・信頼獲得）
　　↓
Webセミナー（30分・AI活用体験）
　　↓
実際のセミナー（対面）
　　↓
AI・ITよろず相談所（個別相談サービス）
```

## 4. ターゲット読者の3層構造

| 層 | 推定割合 | 状態 |
|----|---------|------|
| **A層** | 約30% | AIの名前は知っているが使ったことがない |
| **B層** | 約50%（最重要） | たまに使うが「調べる程度」でとどまっている |
| **C層** | 約20% | 日常的に複数ツールを使いこなしている |

**→ LPはA層・B層をメインターゲットとして設計する**

## 6. デザイン・ブランディングの方針

### カラーパレット（Note記事から踏襲）
| 用途 | カラーコード |
|------|------------|
| メインカラー（濃紺） | `#2C2C5E` |
| アクセントカラー（紫） | `#534AB7` |
| 薄紫背景 | `#EEEDFE` |
| サブカラー（緑） | `#0F6E56` |
| 薄緑背景 | `#E1F5EE` |
| ベージュ背景 | `#F1EFE8` |

### メッセージの方向性
- ✅ 「好きな仕事に使える時間を作る」
- ✅ 「ルーチンワークをAIに任せて、やりたいことに集中する」
- ✅ 「25年のエンジニア経験」「伴走型サポート」
- ❌ 「AIを学ぼう」「ステップアップしよう」（抽象的すぎる）

### トーン
- 親しみやすく、押しつけがましくない
- before/afterの数字で具体性を出す
- 「このおっちゃんに頼めばなんとかしてくれそう」という安心感

## 7. LPに含めるべきコンテンツ要素

1. **ヘッダー**：キャッチコピー＋サービス名
2. **ターゲットの悩み**：「こんな人に」（A層・B層の具体的な悩み）
3. **サービス内容**：よろず相談所でできること
4. **運営者プロフィール**：25年のエンジニア経験、妻に背中を押されたストーリー
5. **実績・before/after**：議事録60分→5分などの数字
6. **相談の流れ**：Facebook/メッセンジャー → Zoom伴走サポート
7. **CTA（行動喚起）**：Facebookグループへの誘導 or メッセンジャー相談
