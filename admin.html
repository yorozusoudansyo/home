記事カード管理

LP の「発信」セクションに表示する記事カードを編集します。
変更内容は data/articles.json に書き出して、GitHub にコミットすることで本番に反映されます。

使い方

  1. 右側のフォームから新しい記事を追加するか、左側の一覧から既存記事を編集 / 並び替え / 削除します。
  2. 「プレビュー」で実際の表示を確認します。変更内容は自動的にこのブラウザに保存されます。
  3. すべて整ったら、ページ下部の 「JSON をダウンロード」 をクリック。
  4. ダウンロードした articles.json を data/articles.json に上書きしてコミット → push（または「GitHub で直接編集」から Web で貼り付けて Commit）
